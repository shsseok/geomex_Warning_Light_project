# 프로젝트 소개
"경광등 테스트 프로그램"은 주어진 환경에서 경광등의 소리와 램프 동작을 테스트하는 프로그램입니다. 
 TCP/IP 프로토콜과 자바 소켓을 이용하여 개발되었으며, 빨강, 초록, 노랑, 파랑, 소리(VOLUME) 등 다양한 테스트 기능을 제공합니다.


## Features

- **Feature 1:** TCP/IP프로토콜 자바 소켓을 이용
- **Feature 2:** 빨강,초록,노랑,파랑,VOLUME 테스트 가능.
